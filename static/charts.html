This page displays the recorded values of Temperature, Humidity, GHI (Global Horizontal Irradiance) and Illuminance in simple line graph. The graph shall be updated in real-time. Users could choose the parameter to be displayed from "Choose a parameter" button, the number of records to be displayed from "Number of records" button, and stop or continue the real-time update with the "Stop/Continue" button. In order to maintain a human-friendly appearance of the graph, the data might be cut off, i.e, the number of records chosen from "Number of records" button might not be the actual number of data points utilized to draw the graph.
